# KeyFlip

> One-keypress fixer for text typed in the wrong Hebrew/English keyboard layout. Windows · AutoHotkey v2.

**תפסיק למחוק. פשוט תהפוך.**

כלי קטן ל-Windows שמתקן טקסט שהוקלד בטעות בפריסת המקלדת הלא נכונה (עברית/אנגלית) — בלחיצה אחת, בלי להקליד הכול מחדש.

## הבעיה

מתחילים להקליד, מרימים מבט — והכול יצא ג'יבריש כי המקלדת הייתה על השפה הלא נכונה. ואז: למחוק, להחליף שפה, להקליד מחדש. שוב ושוב, עשרות פעמים ביום.

## הפתרון

לחיצה אחת על מקש, והטקסט נכתב מחדש בשפה הנכונה — והמקלדת מתחלפת אוטומטית כדי שתמשיך לזרום.

## מה הוא עושה

- **לחיצה אחת** מתקנת את השורה הנוכחית — או בדיוק את הטקסט המסומן.
- **מזהה לבד** את כיוון התיקון: עברית ↔ אנגלית (לפי רוב התווים בקטע).
- **מחליף את שפת המקלדת** בשבילך אחרי התיקון.
- **גם מתג שפה:** לחיצה על אותו מקש בשדה/שורה ריקים מחליפה עברית↔אנגלית מיידית — בלי לפתוח כלום.
- **אייקון במגש המערכת** מראה תמיד את השפה הנוכחית (ירוק = אנגלית, כחול = עברית).
- קליל, רץ בשקט ברקע, וההגדרות נשמרות מקומית.

## התקנה

1. התקינו **AutoHotkey v2**: `winget install AutoHotkey.AutoHotkey` (או מהאתר [autohotkey.com](https://www.autohotkey.com)).
2. הורידו את `KeyFlip.ahk`, `en.ico`, `he.ico` לאותה תיקייה.
3. דאבל-קליק על `KeyFlip.ahk` — אייקון יופיע במגש המערכת.
4. *(רשות)* כדי שירוץ בכל הדלקה — הניחו קיצור דרך לקובץ בתיקיית `shell:startup`.

> **טיפ ל-Windows 11:** אם האייקון לא מופיע במגש, הפעילו אותו ב-Settings → Personalization → Taskbar → "Other system tray icons".

## שימוש

- **ברירת מחדל:** `Alt` + המקש שמתחת ל-Esc → מתקן את השורה (או את הבחירה) ומחליף שפה.
- **שדה/שורה ריקים:** לחיצה על אותו מקש פשוט מחליפה עברית↔אנגלית — מתג שפה מהיר.
- **קליק-ימני** על האייקון ב-Tray → שינוי הקיצור, הפעלה/כיבוי של החלפת-שפה אוטומטית, יציאה.

## הגדרות

נשמרות בקובץ `KeyFlip.ini` ליד הסקריפט (לא נכלל ב-repo — נוצר אוטומטית בהפעלה ראשונה).
